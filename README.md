# Anotações - Conteúdo de Abertura Bootcamp Carrefour Web Developer

## Módulo I

### SCRUM

O Scrum é uma metodologia ágil para desenvolvimento de projetos.

#### Pilares do Scrum

- Adaptação
- Transparência
- Inspeção

#### Sprint

A Sprint é um evento do Scrum, nela ocorrem o planejamento e desenvolvimento das atividades que iram agregar o maior valor possível ao projeto.

#### Papéis e Responsabilidades de um do time

- **_Product Owner_**

  O Product Owner (PO) é responsável pela área de negócios, cabe a ele entender as demandas do cliente e repassa-las ao time de desenvolvimento. O PO é trilha de comunicação entre cliente e o time de desenvolvimento.

- **_Scrum Master_**

  Cabe ao Scrum Master garantir o uso do Scrum, ele é o responsável por reforçar o uso do padrões que o Scrum oferece.

- **_Time de Desenvolvimento_**

  O Time de Desenvolvimento é responsável pelo planejamento e desenvolvimento das atividades a serem seguidas para que uma Sprint seja concluida.
